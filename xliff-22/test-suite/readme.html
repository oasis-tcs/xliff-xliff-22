XLIFF 2.1 Test Suite

Files

  • invalid folder: Invalid core documents, any XLIFF 2.0 reader must be able to detect the error in each document.
  • valid folder: List of valid documents, any XLIFF 2.0 reader must be able to read without error.
  • in-out folder: List of valid input documents ("_in" files) and their corresponding outputs ("_out" files).
